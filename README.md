🌍 Gerçek zamanlı uçuş takibi

✈ Uçak bilgilerini detaylı görüntüleme

🌐 Harita entegrasyonu (Leaflet.js, Google Maps API)

💡 Kullanıcı dostu arayüz

💾 Verileri kaydetme ve analiz etme
